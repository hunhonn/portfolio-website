Academic Projects

Da&Bao

Project Type: Social Entrepreneurship
Year: Oct 2021 - Present

Da&Bao is an environmental sustainability project aimed at eliminating single-use plastics by creating a circular economy of reusable containers in eateries. Initially funded by JP Morgan Chase, Da&Bao has been supported by the SUTD sustainability committee since 2023. The project uses Internet of Things (IoT) technology on campus, integrating all patrons through a locker and a Telegram bot that dispenses takeaway containers. Users are required to return the borrowed containers to tray return areas for washing.

Kastam

Project Type: University Assignment
Year: May 2023 - Aug 2023

Kastam is a border simulator game developed as an assignment for the 40.014 Engineering Systems Architecture module. It was designed to educate users on the importance of understanding queuing systems, leveraging M/M/1 queuing theory. The game was built using RShiny, with interactive components created using ShinyJS to incorporate JavaScript. It is fully integrated with an Amazon Web Services MySQL database to store user and leaderboard data. The game features four levels of increasing difficulty, accounting for 216 different user actions.

Predicting Lightning Strikes

Project Type: University Assignment
Year: Mar 2023 - Apr 2023

For the 40.011 Data and Business Analytics module, I was tasked by the Aviation Studies Institute to predict lightning strikes near ASEAN airports using Machine Learning. Utilizing a dataset from Vaisala in SQL, I also scraped weather data from Don Mueang International Airport using Selenium to enhance the predictive model. Tableau was employed for data visualization to elucidate lightning trends around ASEAN airports. The best-performing model was a binary logistic regression, aligning with Occam’s Razor for its simplicity and effectiveness.

Illuminate Darkness

Project Type: University Assignment
Year: Jan 2022 - Apr 2022

As part of an intra-university competition by JTC, Illuminate Darkness aims to create aesthetic lighting at Changi Business Park using data-driven methods. The project included a fully physical and 1:1 virtual prototype to immerse users. It featured luminous floor panels, a programmable orb, and creatively positioned lighting. The project was nominated for the JTC Awards to be installed in the area. The physical prototype communicated with its virtual counterpart, built in Unity, using File-Transfer-Protocol, enabling synchronized light shows in both environments.
